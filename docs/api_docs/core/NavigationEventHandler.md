# NavigationEventHandler
Namespace: adsk.core
Inherits: EventHandler
Since: March 2021

The NavigationEventHandler is a client implemented class that can be added as a handler to a Navigation event which is triggered by the browser within a palette.

## Methods

### notify(eventArgs: NavigationEventArgs)
The function called by Fusion when the associated event is fired.
- **eventArgs** (NavigationEventArgs): Returns an object that provides access to additional information associated with the event.
