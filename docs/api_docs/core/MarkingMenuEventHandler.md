# MarkingMenuEventHandler
Namespace: adsk.core
Inherits: EventHandler
Since: January 2017

The MarkingMenuEventHandler is a client implemented class that can be added as a handler to a MarkingMenuEvent.

## Methods

### notify(eventArgs: MarkingMenuEventArgs)
The function called by Fusion when the associated event is fired.
- **eventArgs** (MarkingMenuEventArgs): Returns an object that provides access to additional information associated with the event.
