# ActiveSelectionEventHandler
Namespace: adsk.core
Inherits: EventHandler
Since: August 2020

The ActiveSelectionEventHandler is a client implemented class that can be added as a handler to a ActiveSelectionEvent.

## Methods

### notify(eventArgs: ActiveSelectionEventArgs)
The function called by Fusion when the associated event is fired.
- **eventArgs** (ActiveSelectionEventArgs): Returns an object that provides access to additional information associated with the event.
