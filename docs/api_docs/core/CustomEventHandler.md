# CustomEventHandler
Namespace: adsk.core
Inherits: EventHandler
Since: January 2017

The ApplicationEventHandler is a client implemented class that can be added as a handler to an ApplicationEvent.

## Methods

### notify(eventArgs: CustomEventArgs)
The function called by Fusion when the associated event is fired.
- **eventArgs** (CustomEventArgs): Returns an object that provides access to additional information associated with the event.
