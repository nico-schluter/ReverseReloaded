# HttpEventHandler
Namespace: adsk.core
Inherits: EventHandler
Since: January 2024

The HttpEventHandler is a client implemented class that can be added as a handler to an HttpEvent.

## Methods

### notify(eventArgs: HttpEventArgs)
The function called by Fusion when the associated event is fired.
- **eventArgs** (HttpEventArgs): Returns an object that provides access to additional information associated with the event.
