# CameraEventHandler
Namespace: adsk.core
Inherits: EventHandler
Since: December 2017

The CameraEventHandler is a client implemented class that can be added as a handler to an CameraEvent.

## Methods

### notify(eventArgs: CameraEventArgs)
The function called by Fusion when the associated event is fired.
- **eventArgs** (CameraEventArgs): Returns an object that provides access to additional information associated with the event.
