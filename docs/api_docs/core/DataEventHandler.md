# DataEventHandler
Namespace: adsk.core
Inherits: EventHandler
Since: January 2022

The DataEventHandler is a client implemented class that can be added as a handler to a DataEvent.

## Methods

### notify(eventArgs: DataEventArgs)
The function called by Fusion when the associated event is fired.
- **eventArgs** (DataEventArgs): Returns an object that provides access to additional information associated with the event.
