# SetupEventHandler
Namespace: adsk.cam
Inherits: EventHandler
Since: April 2023

The SetupEventHandler is a client implemented class that can be added as a handler to a SetupEvent.

## Methods

### notify(eventArgs: SetupEventArgs)
The function called by CAM when the associated event is fired.
- **eventArgs** (SetupEventArgs): Returns an object that provides access to additional information associated with the event.
